# Filebrowser

Filebrowser is an optional web file manager.

## Why add it
- browse workspace files in a browser
- move through project directories quickly
- useful for light remote admin workflows

## Why it is optional
The core stack does not require a browser file UI.

## Security note
Only expose directories you intend to expose.
Be careful with write access.

## Recommended use
- start with a limited workspace directory
- keep credentials private
- validate locally before any remote access
