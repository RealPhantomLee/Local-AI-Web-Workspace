# Cockpit (Linux Only)

Cockpit is a Linux host administration interface.

## Why it is documented separately
Cockpit is a host-level Linux management tool, not part of the cross-platform core stack.

## Good use cases
- service checks
- logs
- storage and system review
- light remote admin for a Linux machine

## Why it is not part of the base compose stack
- Linux-only
- tied closely to the host system
- outside the cross-platform core scope

## Position in this project
Optional Linux operations tool.
Not required for the local AI workspace core.
