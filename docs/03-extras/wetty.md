# WeTTY

WeTTY is an optional terminal-in-browser interface.

## Why add it
- browser terminal access
- useful for remote operations inside a tailnet
- convenient for quick admin work

## Why it is optional
The core AI workspace does not need browser terminal access.

## Security note
WeTTY increases exposure risk more than a simple dashboard.
Treat it carefully.

## Recommended use
- validate terminal access locally first
- expose through Tailscale Serve only if needed
- avoid public-internet exposure in normal workflows
